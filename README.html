CERBERUS - Cryptographic Engine for Robust, Bulletproof, and Unbreakable Security

CERBERUS is a Python program implementing a cryptographic hash function for robust and secure hashing. The name "CERBERUS" is inspired by the mythical three-headed dog guarding the entrance to the underworld in Greek mythology, emphasizing strong and vigilant security.

Features

  • Robust hashing algorithm
  • Implementation of the CERBERUS cryptographic engine
  • Secure password hashing and verification

Getting Started

To start using CERBERUS, follow these steps:

  1. Ensure you have Python 3.10 || > installed on your machine.
  2. Download the CERBERUS source code from the official repository.
  3. Extract the files to your preferred directory.
  4. Import the XenHash file to your program as follows from CERBERUS import XenHash.
  5. Add the Xen function to your program: Hash = XenHash.Xen(Passw, "XenBits (512 OR 256 OR CR5)")
